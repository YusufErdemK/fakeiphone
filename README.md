A sleek, interactive simulation of an iPhone lock screen, built entirely with HTML, CSS, and JavaScript. Perfect for UI experiments, demos, or just flexing your frontend skills.

---

## Features

![iPhoneSim Screenshot](/screenshots/v0.2.png)

- **Dynamic Notch & Smooth Animations**  
  Responsive hover effects, expanding notch, and elegant transitions.

- **Live Clock & Date**  
  Always up-to-date with the system time, styled for realism.

- **Battery Indicator Simulation**  
  Visual representation with color-coded battery levels.

- **Mini Music Player Integration**  
  Placeholder for Spotify-style mini-player interface.

- **Adaptive UI**  
  Designed to scale subtly on hover for a realistic device feel.

---

## Installation

1. Clone the repository:  
   ```bash
   git clone https://github.com/YusufErdemK/fakeiphone.git
    ```
2. Navigate to the project directory:  
   ```bash
   cd fakeiphone
   ```
3. Open `index.html` in your preferred web browser.
